# Chat App


**A Chat Web-App to Chat and Video call with any user on platform**

This project implements a simple express server that servers http and websocket requests and UI is built using React and used minimum library possible. It features:

* **Chat to any User:** Every user singned up in visble in dashboard.
* **Video Call:** Used Webrtc based library to enable live video call.

**Getting Started:**

1. **Requirements:** NodeJs 18+
2. **Clone Repository:**
    ```bash
    $ git clone https://github.com/tilak-git/ChatApp.gitt
    ```
    
3. **Setup backend server**
    ```bash
    $ cd API
    $ npm i .
    $ npm run dev .
    ```
4. **Setup FrontEnd**
    ```bash
    $ cd FRONT
    $ npm i .
    $ npm start .
    ```
3. **Configure .env and database:**
    * Duplicate sample.env to .env file.
    * Configure environment variables in the .env file.
